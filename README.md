# Iron Arena

Inspired by Diep.io

Iron arena is a game where you play battle royale games, upgrade your tank, and complete the Iron Path, a free battlepass in order to unlock better skins, cosmetics, and even turrets! Find special superpowers and upgrades across the map, and dominate to get victory!

---

In order to run, simply download the file onto your desktop. Then go to Terminal or whatever Command Line it is, and run `cd desktop`. Then run `npm run dev` to open a localhost page. You can then test the prototype from there.
